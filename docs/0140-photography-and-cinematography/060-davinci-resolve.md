# DaVinci Resolve

## Keyboard Shortcuts

### Moving Clips in Track Frame by Frame

- `.` for moving clip to right
- `,` for moving clip to left
- Add `Shift` to move by multiple frames

### Splitting Clips

- `Ctrl + \`
- Use `Alt + \` to join clips back

### Linking Clips

- Select clips
- `Ctrl + L`

### Removing Clips Without Affecting Timeline

- `Backspace`
